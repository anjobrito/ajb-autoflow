# Demo Publicada — AJB AutoFlow V1

## URL

https://ajb-autoflow.vercel.app/

## Status

Demo pública da V1 localStorage publicada na Vercel.

## Observação

Esta versão é demonstrável e usa localStorage no navegador. Os dados cadastrados ficam no navegador do usuário e não representam persistência em banco de dados compartilhado.

## Limites conhecidos

- Sem autenticação real
- Sem PostgreSQL ativo nas telas
- Sem Prisma ativo no fluxo operacional
- Sem Resend obrigatório
- Sem WhatsApp
- Sem Pix
- Sem fiscal/NF-e

## Decisão

A demo está apta para apresentação comercial controlada e portfólio, sem promessa de SaaS produtivo completo.
